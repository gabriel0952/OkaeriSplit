# OkaeriSplit PRD（Product Requirements Document）

## 1. 產品概述

- **產品名稱**：OkaeriSplit（おかえり Split）
- **定位**：簡潔直覺的多人分帳 APP，讓記帳分帳像回家一樣輕鬆
- **目標用戶**：台灣市場優先（中文介面，預設 TWD），架構預留 i18n
- **平台**：Android（API 24+）/ iOS
- **競品參考**：Splitwise（改善其介面複雜、總覽不清晰、建帳流程繁瑣的問題）

## 2. 使用情境

| 情境 | 說明 | 範例 |
|------|------|------|
| 室友/合租 | 長期固定群組，持續記錄日常開銷 | 房租、水電、日用品 |
| 旅行分帳 | 短期活動，多幣別消費 | 機票、住宿、餐飲、門票 |
| 聚餐/活動 | 單次活動費用分攤 | 餐廳、KTV、團購 |

## 3. 核心功能

### 3.1 群組管理

- 建立群組（名稱、類型、預設幣別）
- 邀請成員：邀請碼/連結 + 搜尋用戶帳號
- 群組成員管理（加入/退出）

### 3.2 記帳

- **基本資訊**：金額、付款人、日期、備註
- **消費分類**：餐飲、交通、住宿、娛樂、日用品、其他（可自訂）
- **附件**：收據/照片拍照上傳
- **多幣別**：支援不同貨幣記帳，可設定匯率

### 3.3 分帳方式

- **均分**：所有參與者平均分攤
- **自訂比例**：按比例分配（如 2:1:1）
- **指定金額**：手動指定每人金額
- **項目拆分**：單筆消費中個別項目指定由誰付（如餐廳各點各的）

### 3.4 結算

- 欠款總覽：誰欠誰多少，一目了然
- **最簡化轉帳**：自動計算最少轉帳次數（債務簡化演算法）
- 手動標記已付款
- 結算歷史記錄

### 3.5 帳號與同步

- 註冊/登入（Email + Google Sign-In + Apple Sign-In）
- 雲端資料同步（Supabase）
- 多裝置支援

### 3.6 總覽與統計

- 個人總覽：我欠別人 / 別人欠我的彙總
- 群組消費統計（按分類、按時間）
- 清晰的 Dashboard 設計（改善 Splitwise 痛點）

## 5. 設計原則

- **簡潔優先**：新增消費常用路徑 ≤ 3 步完成（金額 → 描述 → 送出），進階功能以漸進揭露收折
- **一眼看懂**：Dashboard 讓使用者立刻知道自己的帳務狀態
- **快速操作**：常用功能（記帳、查欠款）要能最快到達
- **視覺品質**：Apple 冷白系設計語言——乾淨底色、精緻字距、卡片以背景色差代替陰影

### 5.1 視覺設計方向

- **風格定位**：Apple iOS/macOS 現代簡約風，非 Material 原生感
- **色彩系統**：Indigo 主色 (#4F46E5)，淺灰底色 (#F5F5F7)，白色卡片，語義色（綠=應收、紅=應付）
- **新增消費**：Progressive Disclosure——金額大字顯示在頂部，分帳方式與進階選項預設收折，避免視覺過載

### 5.2 新增消費互動原則

- **金額優先**：頂部以大字 display 呈現，點擊後彈出數字鍵盤（僅接受數字與小數點）
- **智慧預設**：幣別沿用群組預設、日期預設今天、分攤成員預設全選、分帳方式預設均分
- **漸進揭露**：分帳方式（非均分）與進階選項（日期、備註、附件）預設折疊，降低初次使用門檻
- **狀態同步**：編輯模式且已有備註/附件時，「更多選項」區塊自動展開，確保資料可見

## 6. 技術棧

> 詳見 `CLAUDE.md`

- Flutter 3.x / Dart 3.x
- 狀態管理：Riverpod
- 導航：go_router
- 後端：Supabase
- 本地儲存：Hive
- Data class：freezed
- 錯誤處理：fpdart (Either)

## 7. 非功能需求

- **離線可用**：本地 Hive 快取，上線後同步
- **效能**：列表滑動 60fps，啟動 < 2 秒
- **安全**：敏感資料加密儲存，API 通訊 HTTPS
